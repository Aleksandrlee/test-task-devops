# Тестовое задание для DevOps-инженера

## Описание задания
В рамках тестового задания был создан проект с использованием `Docker` и `docker-compose`, включающий следующие сервисы:
- **Django** (Python 3.11) — основное приложение.
- **PostgreSQL** (версия 16 на Alpine Linux) — база данных.
- **Frontend** — условный фронтенд, работающий на порту 3000.

Кроме того, была настроена система `healthcheck`, чтобы обеспечить порядок запуска контейнеров:
1. `Postgres` должен запуститься и быть готов к работе, прежде чем запустится `Django`.
2. `Frontend` должен запуститься только после стабильной работы `Django`.
